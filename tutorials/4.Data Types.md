Numbers can be separated with a an underscore `_` as a visual separator so numbers are easier to read such as `1_000_000`.
Numbers can be represented under different bases wich are:

| Base    | Example     |
|---------|-------------|
| Decimal | 42          |
| Hex     | 0xAF        |
| Octal   | 0o53        |
| Binary  | 0b0110_1110 |
